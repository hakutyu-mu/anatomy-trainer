# Anatomy National Exam Trainer

解剖学の国家試験対策のための、4択クイズWebツールです。
スマートフォン・PCの両方で快適に学習でき、オフライン（ローカル）環境でも動作します。

## 🚀 特徴
- **3つの学習モード**: ランダム出題、順番に出題、復習モード（間違えた問題のみ）。
- **中断・再開機能**: クイズを途中で中断しても、次回続きから再開できます。
- **学習ログ・カレンダー**: 日々の回答数や正答率をカレンダー形式で可視化。
- **完全ローカル動作**: `standalone.html` を開くだけで、ブラウザがあればどこでも動作します。

## 📂 構成
- `index.html`: 標準版（外部JS/CSS読み込み）
- `standalone.html`: 全機能を1ファイルに統合したポータブル版
- `css/`: スタイルシート
- `js/`: 各種制御ロジック（ストレージ、音声、クイズ、カレンダー）
- `data/`: 問題データ（JSON形式およびJS定数形式）

## 🛠 使い方
1. リポジトリをダウンロードまたはクローンします。
2. `standalone.html`（または `index.html`）をブラウザで開きます。
3. 学習を開始します。データはブラウザ（LocalStorage）に保存されるため、ページを閉じても記録は残ります。
